# Project Setup Guide

## Quick start with Docker

- `docker compose up -d`
- Frontend runs on: http://localhost:5173
- Backend runs on: http://localhost:3000

## Running the client

- `cd client`
- `pnpm i`
- `npm run dev`
- Frontend runs on: http://localhost:5173

## Running the server

- `cd server`
- `pnpm i`
- `npm run dev`
- Backend runs on: http://localhost:3000

The database is empty by default. You can add a test user by running index.ts
